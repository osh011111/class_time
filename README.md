# 강의실 시간 배정 관리 및 기록 시스템

리눅스 Shell Script와 C언어를 연동하여 교실 사용 시간을 효율적으로 배정하고 배정 기록을 저장하는 프로그램입니다.
또한 날짜를 기반으로 기록을 확인할 수도 있습니다.

## 주요 기능
1. **시간 배정 요청**: 사용자가 희망 시간대(예: `10-12`)를 입력하면 C 알고리즘을 통해 스케줄링 결과를 계산합니다.
2. **로그 자동 저장**: 모든 실행 결과는 타임스탬프와 함께 `schedule.log` 파일에 안전하게 기록됩니다.
3. **과거 이력 검색**: 저장된 로그 파일에서 특정 날짜의 기록만 빠르게 찾아볼 수 있습니다.
4. **자동 컴파일**: 프로그램 실행 시 `scheduler.c` 소스 코드를 자동으로 감지하고 컴파일합니다.

## 실행 방법

**실행 권한 부여**
터미널을 열고 스크립트 파일에 실행 권한을 줍니다.
chmod +x manager.sh

**프로그램 실행**
./manager.sh

**메뉴 1번 (시간 배정):** 시작시간-종료시간 형식으로 입력 (예: 09-11 13-15)

**메뉴 2번 (기록 검색):** YYYY-MM-DD 형식으로 입력 (예: 2025-11-26)
